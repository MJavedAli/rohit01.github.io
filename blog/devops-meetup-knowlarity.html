Devops Meetup @ Knowlarity

By Rohit Gupta on Feb 21, 2015, 09:00 pm (IST)

Today, we hosted a DevOps Bangalore Group meetup at Knowlarity office. A picture can tell a thousand words. So, let’s enjoy the story of this event with a slideshow (photo courtesy - @soumpaul).

Tech Talks

We had a bunch of Devops related tech talks delivered by well known speakers from IT industry. Here’s the schedule with presentation slides:

  • 11:00 AM: RIP Nagios. Hello Docker Shinken!
    Speaker: Rohit Gupta (me)
    Synopsis: RIP Nagios. Hello Docker Shinken!
    Slides link: RIP Nagios. Hello Docker Shinken!

  • 11:30 AM: Salt-Stack for Configuration Management

    Slides link: Salt Stack

  • 12:00 PM: Analyzing logs using ElasticSearch, Fluentd, Kibana
    Speaker: Ravdeep Pasricha
    Twitter: @ravdeep003
    Slides link: Analyzing logs using EFK

  • 12:30 PM: Lunch & Networking
    Sponsored by Knowlarity

  • 01:00 PM: Apache Falcon (Data Pipeline Onboarding)
    Speaker: Sanjeev Tripurari
    Slides link: Apache Falcon

  • 01:30 PM: Puppet for Configuration Management(Demo)
    Speaker: Gourav Shah
    Twitter: @gouravshah
